# Agentic Trajectory and Growth Plan

## Stage 1: Foundation (Current)

- Contract-first specs and governance policies
- Planner/Worker/Judge skeleton with immutable models
- Failing TDD contracts established

## Stage 2: Controlled Autonomy

- Integrate MCP-driven trend ingestion into planner decomposition
- Integrate MCP-driven social publishing and reply execution into worker flow
- Govern queue execution with orchestrator + judge status transitions
- Implement concrete runtime skills and retry policy
- Add confidence calibration telemetry

## Stage 3: Economic Agency

- Introduce wallet providers behind strict budget controls (Coinbase provider + simulated fallback baseline)
- Transaction policy simulation, then controlled enablement

## Stage 4: Fleet Scaling

- Horizontal worker autoscaling
- Tenant-aware throughput controls
- Advanced observability and anomaly detection
